关于英雄联盟数据分析平台

项目简介

这是一个基于Flask和GitHub Pages的英雄联盟数据分析平台，采用双站协作架构， 结合了PythonAnywhere的动态处理能力和GitHub Pages的静态展示优势。

技术架构
  • 后端: Flask + SQLite + Pandas
  • 前端: Bootstrap 5 + Chart.js
  • 部署: PythonAnywhere + GitHub Pages
  • 自动化: Git + 定时同步脚本
功能特点
  • 装备数据查询与分析
  • 比赛数据统计与可视化
  • CSV文件上传与处理
  • 数据导出与分享
  • 响应式网页设计
  • 双站自动同步
